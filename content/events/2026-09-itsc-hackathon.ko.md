IEEE ITSC 2026 조직위원회는 이탈리아 나폴리에서 개최되는 ITSC 2026 기간 중 **IEEE ITSS Hackathon**을 개최합니다. 본 해커톤의 주제는 **"ITSS Smart Mobility Data Challenge"**(ITSS 스마트 모빌리티 데이터 챌린지)입니다. 혁신가, 학생, 전문가, ITS 애호가 여러분의 많은 참여를 부탁드립니다.

본 경연은 AWS 클라우드 서비스 기반의 4시간 데이터 중심 행사로, 참가자들은 복합(heterogeneous) "Data Hypercube"를 탐색하며 도시 모빌리티의 실제 문제에 대한 해결책을 탐색할 것입니다. 고급 프로그래밍 능력은 필요하지 않으며, 데이터 분석과 KPI 최적화에 초점을 맞춥니다.

## 행사 정보

| | |
|---|---|
| **형식** | 4시간 현장 경연 |
| **플랫폼** | AWS 클라우드 (시각화 분석 도구 및 컴퓨팅 환경 제공) |
| **장소** | 이탈리아 나폴리 — ITSC 2026 기간 중 (2026년 9월 15–18일) |
| **정원** | 선착순 100명 |
| **참가 자격** | IEEE ITSC 2026에 등록한 학생, 연구자, ITS 애호가 |

## 해커톤의 의의

- 모빌리티 데이터의 폭발적 증가에 따라 새로운 관리 솔루션이 요구됨
- 4시간의 실습을 통해 수 주간의 이론 학습 이상의 경험 획득
- 객관적 평가 — 심사위원의 주관적 편향 배제

## 예시 챌린지

"ITSS Smart Mobility Data Challenge"는 다음과 같은 과제를 다룹니다:

- **대중교통(LPT):** 노선, 정류장, 버스·지하철 시간표 등 네트워크 구조
- **도로 교통:** 택시·물류 등 차량 함대의 익명화된 GPS 데이터로부터 평균 속도와 혼잡도 지도화
- **마이크로 모빌리티:** 환경 영향 완화 전략

## 채점 시스템

주관적 심사 대신 객관적 최적화 기반:

- 참가자는 코드가 아닌 솔루션(예: 좌표 목록, 스케줄 행렬)을 제출
- 솔루션은 시뮬레이션을 수행하는 엔드포인트로 전송
- 시뮬레이터가 KPI 점수를 계산해 반환
- **실시간 리더보드**가 순위를 즉시 갱신
- 동점 시에는 가장 짧은 시간에 최고 점수를 달성한 팀이 우승

## 시상

IEEE는 상위 3개 팀에게 상을 수여합니다. 모든 참가자에게는 참가 인증서가 발급됩니다.
